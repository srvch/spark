# Spark iOS

Native SwiftUI scaffold for Spark, designed as a premium iPhone app for discovering and creating nearby nano events.

## Open in Xcode

This repo includes Swift source and an `XcodeGen` spec at `project.yml`.

1. Install Xcode.
2. Install XcodeGen if needed.
3. From this folder run `xcodegen generate`.
4. Open `SparkIOS.xcodeproj` in Xcode.

## Notes

- Target platform: iOS 17+
- Built with SwiftUI
- Includes a motion-first discovery screen and a create-spark flow
